# Stroop Effect Test

The Stroop Effect Test application assesses the cognitive interference that occurs when the name of a color is printed in a color not denoted by the name. This phenomenon demonstrates the difficulties our brain encounters when trying to process conflicting information.

## Features
- Real-time testing of reaction times.
- Interactive UI for engaging users.
- Easy to understand instructions.

## Usage
1. Launch the application.
2. Start the test and follow prompts.
3. Color words will appear, and you should select the color of the text, not the word itself.

## Conclusion
The Stroop Effect Test serves as a fascinating tool for exploring cognitive psychology principles and can be utilized in various research settings or educational contexts.
